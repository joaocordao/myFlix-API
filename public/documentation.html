API Documentation

Operation Request URL Request Body Response
Return a list of ALL movies GET /movies List of movies
Return data about a single movie by Title GET /movies/:title N/A Data of the movie
Return data about a genre by name GET /movies/genre/:genreName N/A Data of the genre
Return data about a director by name GET /movies/director/:directorName N/A Data of the director
Allow new users to register POST /users { "Username": "example", "Password": "password", "Email": "example@example.com", "Birthday": "1990-01-01" } Confirmation message
Return a list of ALL Users GET /users N/A List of users
Allow users to update their user info PUT /users/:userId { "Username": "newUsername", "Password": "newPassword", "Email": "new@example.com", "Birthday": "1995-01-01" } Confirmation message
Return data about a user by username GET /users/:Username N/A Data of the user
Allow users to add a movie to their list of favorites POST /users/:username/movies/:movieID N/A Confirmation message
Allow users to remove a movie from their list of favorites DELETE /users/:username/movies/:movieId N/A Confirmation message
Allow existing users to deregister DELETE /users/:username N/A Confirmation message
